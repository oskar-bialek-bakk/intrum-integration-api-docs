# API Integracyjne DEBT Manager

Dokumentacja techniczna API integracyjnego systemu DEBT Manager. API umożliwia automatyczny import danych od kontrahentów, zarządzanie kolejkami komunikatów oraz monitorowanie statusu importów.

- **[Założenia](zalozenia/index.md)** — konfiguracja kontrahentów, procedura importów, architektura fizyczna, kolejki
- **[Funkcje API](funkcje-api/index.md)** — opis metod API (importy, statusy, słowniki, walidacje)
- **[Komunikaty](komunikaty/index.md)** — specyfikacja komunikatów wymiany danych (Case, Customer, Contract, Payment i inne)
- **[Przypadki użycia](przypadki-uzycia/index.md)** — praktyczne scenariusze użycia API
